# spec.md — free-worker

生成日: 2026-06-09 / 出典パック: `docs/reference/`(kojin_freelance_ec_subscription_startup_pack_20260609)

## 目的
日本在住の個人事業主・フリーランス(EC・デジタルコンテンツ・サブスク・受託)が、開業準備〜日常運用の事務(法令チェック・商品/サブスク管理・売上/請求・経費・契約/規約・コンテンツ制作・顧客同意・バックアップ)を**1つのローカルアプリ**で管理する。クラウドAI/外部APIが使えなくても全機能が動く「保険」。

## 対象ユーザー / 非対象
- 対象: 個人事業主(予定者含む)、フリーランス、屋号レベルの個人経営。
- 非対象: 会計ソフトの完全置換、e-Tax連携、決済代行、法務/税務の最終判断(専門家確認前提)。

## 絶対要件 (アーキ不変条件)
1. **ローカルファースト:** SQLite + ローカル添付 + ローカル暗号化バックアップ。サーバは `127.0.0.1` のみ。
2. **AI非依存:** AIなしで全機能が動く。AI補助(要約等)は任意アダプタ(Ollama/LM Studio/Gemini Flash無料枠)で既定OFF。
3. **機密を外に出さない:** 実顧客データ・税番号・契約原本をクラウドへ送らない。開発は合成データのみ。
4. **オフライン動作:** ネット遮断状態で起動・主要CRUD・検索・CSV出力・バックアップ/復元が成功する。

## MVP (Must) と受け入れ条件
- **EPIC-01 事業プロフィール:** 屋号/住所公開方針/税務署/開業日/申告方式/インボイス登録を保存・編集でき、期限計算に使える。
- **EPIC-02 法令・税務チェックリスト:** 各項目に期限・証跡ファイル・出典URL(S1..S15)・状態を持てる。`action_checklist.csv` をシードに使う。
- **EPIC-03 商品・サブスク管理:** SKU/価格(税込)/ライセンス/動作環境/サブスクプランを管理。**商品ページ掲載に必要な項目の欠落を警告**する(特商法/動作環境/継続課金条件)。
- **EPIC-05 売上・請求:** 注文/入金/返金/請求書/適格請求書項目を管理。**CSVインポート/エクスポート + 月次集計**ができる。
- **EPIC-10 暗号化ローカルバックアップ:** 暗号化バックアップを作成/復元でき、**復元テストログ**が残る。
- **EPIC-11 AI非依存モード:** ネットワーク無効状態でも主要操作が成功する(自動テストで保証)。

## Should
- EPIC-04 契約・規約の版管理 / EPIC-06 経費・証憑 / EPIC-07 コンテンツ制作WF(公開前チェック) / EPIC-08 顧客・同意 / EPIC-12 監査ログ(CSV出力)。

## Could
- EPIC-09 ダッシュボード(MRR/売上/未収/返金率/解約率/粗利/今月の期限)。

## アーキテクチャ
- `@free-worker/core`: ドメイン(DB/migration/services/csv/backup/money/dates/audit/ai-adapter IF)。ヘッドレスで vitest 可能。
- `@free-worker/server`: Fastify、`127.0.0.1` bind、REST API、`web/dist` を静的配信。
- `@free-worker/web`: Vite+React+TS SPA。

## 完了の定義
ネット遮断でアプリ起動 → 主要CRUD・検索・CSV出力・バックアップ/復元が動く。typecheck/test/build green。実顧客データをクラウドへ送っていない。`docs/compliance_sources.md` に出典と確認日が残る。
