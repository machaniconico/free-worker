# EC・デジタルコンテンツ・サブスク用 文面設計メモ

生成日: 2026-06-09  
注意: これは実務メモです。最終公開前に、実際の商品・販売形態・居住地・公開住所方針に合わせて専門家確認してください。

## 1. 特定商取引法に基づく表記 - 項目案

- 販売業者: 個人事業主名または屋号。氏名・住所・電話番号の公開方針は、特商法ガイドとプラットフォーム仕様を確認。
- 運営責任者: 個人の場合は本人名を想定。
- 所在地: 公開住所。自宅公開を避ける場合はバーチャルオフィス等を検討。ただし、実態・郵便受取・法令適合を確認。
- 電話番号: 連絡可能な番号。問い合わせ窓口の対応時間も併記。
- メールアドレス/問い合わせフォーム。
- 販売価格: 税込表示。商品ページと決済画面で一致させる。
- 商品代金以外の必要料金: 決済手数料、通信費、振込手数料など。
- 支払方法: クレジットカード、銀行振込、プラットフォーム決済等。
- 支払時期: 申込時、更新日、無料期間終了後など。
- 引渡時期/提供時期: ダウンロードURL発行、会員権限付与、講座公開日など。
- 返品・キャンセル・解除: デジタル商品の性質、サブスク解約方法、解約後閲覧、返金条件。
- 動作環境: ソフト、テンプレ、ツール、素材集はOS/アプリ/バージョン/必要環境。
- 継続課金条件: 契約期間、更新周期、更新価格、解約期限、最低契約期間の有無。

## 2. 利用規約 - 章立て案

1. 適用範囲
2. アカウント登録と管理
3. 商品/会員サービスの内容
4. 料金、支払い、更新、解約
5. デジタル商品のライセンス
6. 禁止事項
7. 知的財産権
8. 投稿・コミュニティ機能がある場合のルール
9. サービス停止・変更・終了
10. 返金・キャンセル
11. 免責・責任制限
12. 準拠法・管轄
13. 規約変更の通知

## 3. プライバシーポリシー - 項目案

- 取得する情報: 氏名、メール、決済識別子、注文履歴、問い合わせ内容、アクセスログ等。
- 利用目的: 商品提供、本人確認、課金、問い合わせ対応、メール配信、改善、不正防止、法令対応。
- 第三者提供: 原則なし。法令、決済、配送/委託、プラットフォーム利用時を明記。
- 委託先: 決済代行、ECプラットフォーム、メール配信等。
- 保管期間: 税務・契約・問い合わせ対応の必要期間。
- 安全管理: アクセス制限、2FA、バックアップ、暗号化、権限管理。
- 開示・訂正・利用停止等の窓口。
- Cookie/解析ツール/広告タグの利用。

## 4. デジタル商品のライセンス条項 - 商品別に決める項目

- 利用者: 購入者本人のみ / チーム可 / 商用利用可否。
- 禁止: 再配布、転売、改変再配布、生成AI学習への投入、第三者共有など。
- 成果物への組込み可否。
- クレジット表記要否。
- アップデート提供範囲。
- 返金条件。
- 違反時のアカウント停止・損害賠償。
